### 1. 新建github仓库
> 基于github page搭建

新仓库名要求： `{{username}}.github.io`

### 2. 安装hexo
全局安装hexo
```
npm i -g hexo-cli
```
初始化博客文件夹
```
hexo init {{folder}}
```
进入文件夹后
```
hexo server
```
默认启动在 localhost:4000 就已经有第一个页面啦


## 常用命令
